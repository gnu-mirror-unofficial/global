Known bugs

Known bugs in the latest GNU GLOBAL.

  • Gtags cannot detect function pointer correctly. [report]
  • Gtags cannot detect macro references in assembly source code. [report]
  • Defined macro not found. [report]
  • Nested functions not found. [report]
  • Global with the -x option abort in a special case. [report]
  • It seems like derived structures is not supported. [report]
  • backtics operator in PHP is not recognised as string delimiter [report]
  • Global seems to lack support for c++'s "using namespace <id>" [report]
  • template typedefs not handled properly [report]
  • always getting "Warning: ... is out of source tree.(Windows)" [report]
  • C++ parsing problem with constructors and destructors. [report]
If you can fix the bugs then please send it to us. Thank you in advance.
