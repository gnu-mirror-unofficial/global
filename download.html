Getting GLOBAL

Source archive

You can download the latest source archive from here or GNU ftp site.

CVS repository

You can browse the GLOBAL CVS repository via the CVSweb interface.

You can also use the CVS utility to obtain anonymous read-only access. The module name is global.

You can get and build CVS version of GLOBAL like this:
(You need automake, autoconf, gperf and flex installed in your machine.
Hit return when prompted for the login password.) 

% cvs -d:pserver:anoncvs@subversions.gnu.org:/cvsroot/global login
(Logging in to anoncvs@subversions.gnu.org)
CVS password: [RET]
% cvs -d:pserver:anoncvs@subversions.gnu.org:/cvsroot/global co -P global
% cd global
% sh reconf.sh 
% ./configure
% make
