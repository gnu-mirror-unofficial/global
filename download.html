Getting GLOBAL

Source archive

You can download the latest source archive from global-6.2.9.tar.gz or GNU ftp site.
The archive files in GNU ftp site are signed by Shigio YAMAGUCHI, the release manager of GNU GLOBAL. Here is Shigio's PGP public key (GPG Key).

Binary archive

GNU GLOBAL project only distributes source package, and GNU GLOBAL only supports UNIX(POSIX) environment. The following packages are due to outside projects.

DOS and Windows 32 version are distributed by Jason Hood.

Mac OSX version is distributed by MacPorts .

AIX version and HP-UX version are also available.

You can also get binary package from following distributions:

  • Fedora GNU/Linux.
  • Debian GNU/Linux <- It's too old. Please don't use Debian's package.
  • FreeBSD.
  • NetBSD.

CVS repository

You can browse GLOBAL CVS repository via the ViewCVS interface.

You can also use CVS utility to obtain anonymous read-only access. Please see savannah CVS page for the details.

You can build CVS version of GLOBAL like this:
(You need automake, autoconf, gperf, bison and flex installed in your machine.) 

% sh reconf.sh 
% ./configure
% make
